Cosa fa

Gestione soci

Crea, modifica, rinnovi ed elimini i soci del club. Anagrafica, numero tessera, scadenza annuale.

Tessere QR

Genera tessere con QR code permanente per ogni socio. Il QR identifica la persona, non scade al rinnovo.

Scanner presenze

Registra gli ingressi scansionando il QR della tessera, oppure cercando il socio per nome o numero.

Riepilogo e report

Visualizza le presenze mensili, esporta PDF e CSV. Selettore mese per consultare lo storico.

Backup e export

Esporta backup completi (JSON), fogli di calcolo (CSV) e tessere PDF. Tutto sul tuo filesystem.

Sicurezza

Password con hash PBKDF2-SHA512, sessioni protette, CSP. Nessuna telemetria, nessun tracking.

Sicurezza per design

L'app gestisce dati di soci reali. La sicurezza non è un'aggiunta, è il punto di partenza.

Password irrecuperabili

Hash PBKDF2-SHA512 con 310.000 iterazioni e sale casuale. Nessuno può risalire alla password originale, nemmeno con accesso al database.

Sessioni protette

Token a 32 byte hashati, cookie HttpOnly + Secure + SameSite. Rotazione automatica al cambio password. Nessuna sessione persistente esposta.

Rate limiting

Blocco automatico dopo tentativi falliti. Login, recupero password e azioni admin hanno limiti indipendenti. Anti-enumerazione con tempi costanti.

CSP e capability

Content Security Policy rigorosa nel bundle Tauri. Capability esplicite con permessi minimi. Nessuna esecuzione di codice esterno.

Zero telemetria

Nessun analytics, nessun tracking, nessun invio di dati. L'unica comunicazione di rete è il controllo aggiornamenti verso GitHub.

Codice verificabile

Open source MIT. Chiunque può auditare il codice, verificare le implementazioni e contribuire con miglioramenti di sicurezza.

Domande frequenti

Dove finiscono i dati?

Sul tuo computer. Nella versione desktop, tutto viene salvato in un file locale nella cartella dati dell'app. L'unica comunicazione di rete è il controllo aggiornamenti verso GitHub (nessun dato personale inviato).

Serve internet?

No. L'app funziona completamente offline. L'unica comunicazione di rete è il controllo aggiornamenti verso GitHub, che avviene all'apertura e non invia dati utente.

Come faccio il backup?

Dalle impostazioni admin puoi esportare backup completi (JSON) o fogli di calcolo (CSV). I file finiscono nella cartella Download. Conservali su un supporto sicuro.

La fotocamera non funziona

Verifica che il permesso fotocamera sia attivato nelle impostazioni di sistema. Su Windows: Impostazioni → Privacy → Fotocamera. Su macOS: Impostazioni di Sistema → Privacy → Fotocamera. In alternativa, usa la ricerca manuale per nome o numero tessera.

Posso usarla su più computer?

L'app è progettata per uso locale su un singolo computer. Per usare gli stessi dati su un altro computer, esporta il backup dal primo e ripristinalo sul secondo.

È sicura?

Le password sono hashate con PBKDF2-SHA512 (310.000 iterazioni). Le sessioni usano token hashati con cookie HttpOnly e SameSite. C'è una Content Security Policy. Nessuna telemetria. Il codice è open source e verificabile.
